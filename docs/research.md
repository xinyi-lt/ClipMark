# 浏览器高亮插件竞品调研

## 结论

第一版应避开成熟产品的“大而全”路线，聚焦本地、轻量、Markdown-first。核心差异点不是知识库规模，而是把网页高亮和备注快速变成可复制、可沉淀的 Markdown。

## 竞品摘要

### Glasp

- 优势：网页/PDF 高亮、分享、Obsidian/Notion/Readwise 集成、AI 总结、社交发现。
- 借鉴：高亮后快速沉淀到外部知识系统。
- 不跟进：社交网络、云端账户、AI。

### Readwise Reader

- 优势：统一阅读库、稍后读、跨设备同步、导出到知识库工具。
- 借鉴：导出稳定性和阅读流完整性。
- 不跟进：订阅型阅读系统、完整内容库。

### Hypothesis

- 优势：公开/私有群组批注、教育场景、网页/PDF/视频/图片标注。
- 借鉴：注释与原文紧密绑定。
- 不跟进：协作、群组、LMS。

### Weava

- 优势：网页/PDF 高亮、颜色、注释、文件夹、引用。
- 借鉴：颜色和注释的基本研究工作流。
- 不跟进：复杂资料库、引用管理。

### Diigo

- 优势：老牌书签、高亮、便签、归档和知识库。
- 借鉴：sticky note 与页面级组织。
- 不跟进：云端 library。

### MarkDownload

- 优势：网页到 Markdown 的复制和下载体验清晰。
- 借鉴：Markdown 导出是核心路径。
- 不跟进：完整网页正文转换，MVP 只导出高亮和备注。

## MVP 范围

- 支持 Chrome / Edge MV3。
- 支持普通网页文字选区高亮。
- 支持每条高亮的注释编辑和删除。
- 支持 popup 查看当前页高亮列表。
- 支持复制当前页 Markdown。
- 支持下载当前页 `.md` 文件。
